Insructions Find below the instructions to use this website


This webpage was created to represent the Covid-19 data in a graph format for the easy understanding

of the disease spread.


The webpage contains the following categories in the main menu:-


  • Interactive Graph An interactive graph of the same data which can be dragged.


  • Add Person: A form to add a Person details to the database(Person ID, District, Status).


  • Create Link: A form to add two Person ID to establish a link between them(Person1 ID, Person2 ID).


  • Contact: Find our contact details here in case required.
